Podstawowe znaczniki html

  • <html> - korzeń dokumentu
  • <head> - nagłówek strony z infromacjami dla przegladrek
  • <body> - właściwa treść dokumentu widoczna w knie przeglądarki
  • <div> - element blokowy
  • <p> - akapit
  • <ol>, <ul> i <li> - elementy listy wypunktowanej lub numerowanej
  • <img> - obrazek
  • <a> - hiperłącze
